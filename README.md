## Linux 指令

https://dylan237.github.io/linux-basic.html

 * 創資料夾

    mkdir (-p) Name

-p:直接建立多層的資料夾

 * 移除檔案

    rm (-r)(-f) fileName

-r 刪除資料夾

-f 強制刪除 ,-rf=資料夾連同檔案刪除

 * 顯示目錄
   
    ls (-a)(-l)(-r) (DIR)

-a 顯示隱藏檔案

-l 詳細資料 （權限、日期）

-r 顯示包含子資料夾

DIR 沒有，默認當前目錄
